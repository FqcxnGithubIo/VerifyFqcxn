Verification Information

  • Why Verify?

    In case of any raids, nukes, etcetera, that damages our server, we have backups of everything and can be restored to the original state. These backups include:

    Discord Server Categories | Discord Server Channels | Discord Server Roles | Discord Server Verified Memebers | Discord Member Roles | ETC.


    What Happens When You Verify?

    When you verify, you will automatically gain access to the rest of the discord server. Meaning, you can chat and view channels that non-verified users can't see.


  • Saved Discord Server Data

    We backup our Discord Server's categories, channels, roles, and members. Backups will be loaded whenever an emergency happens. (Raids, Nukes, ETC.)

    - Categories : Discord Server Categories and Their Set Permissions
    - Channels : Discord Server Channels and Their Set Permissions
    - Roles : Discord Server Roles and Their Set Permissions
    - Members : Discord Server Member's; Roles, Verified Members


    What Information is Passed to us when Verifying?

    - Your Discord ID, Username, Profile Picture, Banner
    - Your Email Address
    - Third Party Connections
    - Your IP Address
